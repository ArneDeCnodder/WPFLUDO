## omschrijving: 
- Voor het vak WPF zou ik graag het klassieke bordspel ‘Mens Erger je niet’ digitaliseren. 
Ik heb tijdens de lockdown een eigen versie van het klassieke bordspel gemaakt omdat 
het nog steeds een van men favorieten is. Het nadeel van mijn creatie was dat het op 
papier was en het online spelen van deze variant niet zo aangenaam was. Daarom zou 
het leuk zijn als ik dit klassieke bordspel kan digitaliseren.
